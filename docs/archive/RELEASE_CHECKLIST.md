# Graph Atlas 发布验收清单

## 1. 数据与持久化

- [ ] 损坏 localStorage 不会导致页面崩溃。
- [ ] 新增节点刷新后仍存在。
- [ ] 重置功能有二次确认。
- [ ] 示例数据可恢复。
- [ ] 旧 `graph-atlas-nodes` 数据有迁移或回退策略。

## 2. 图谱与关系

- [ ] 图谱连线和详情关系列表来自同一份 `edges`。
- [ ] 修改节点标题不会破坏关系跳转。
- [ ] 关系目标不存在时有降级处理。
- [ ] 默认护照关系完整显示。

## 3. 隐私与安全

- [ ] 所有实体有隐私级别。
- [ ] 高隐私资料默认 `aiAccess: false`。
- [ ] UI 不再误导性声称“已加密”。
- [ ] AI 上下文预览排除高隐私资料。
- [ ] 隐私切换有清晰说明。

## 4. 搜索与资料查找

- [ ] 搜索覆盖标题、类型、标签和内容。
- [ ] 搜索空状态有新增入口。
- [ ] 护照附件可通过搜索或详情找到。
- [ ] 关联节点可跳转。

## 5. 新增资料

- [ ] 标题必填。
- [ ] 类型和隐私级别必填。
- [ ] 新增后自动选中。
- [ ] 新增后可持久化。
- [ ] 新增资料可建立关系。

## 6. 附件与来源

- [ ] 原有附件能正常显示。
- [ ] 无附件状态清晰。
- [ ] 来源证据或手动创建标记可见。
- [ ] 无来源状态提示补充证据。

## 7. 视觉与体验

- [ ] 默认桌面视图无明显重叠。
- [ ] 移动宽度下侧边栏隐藏后核心功能可用。
- [ ] 详情面板长内容可滚动。
- [ ] 所有可点击按钮有明确结果或禁用状态。

## 8. 发布前文档

- [ ] `PRODUCT_SPEC.md` 与实际范围一致。
- [ ] `MVP_PRD.md` 的验收项已逐项检查。
- [ ] `PRIVACY_MODEL.md` 与 UI 文案一致。
- [ ] `DEVELOPMENT_BACKLOG.md` 已标记已完成和延期项。
